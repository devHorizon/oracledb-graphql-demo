About Oracle DB

Relational Databases

  • Query Language is SQL
  • Made up of "Tables" (think spreadsheets)
  • Each table has "Columns" (attributes) and "Rows" (instances)
  • "Schema on Read" (Structured Data for high performance)

Our Apps RDBS Schema

This schema might look slightly different than the one in About GraphLQ but don't be alarmed! This is just how the database sees the data but this defines the same schema. The only difference here is the language behind the definitions. Oracle Database uses SQL which is the gold-standard of data definition and manipulation languages.

SQL has an interesting history. It was originally designed by IBM to facilitate interaction with relational data, and named SEQUEL (Structured English Query Language). It wasn't until Relational Software, Inc (now Oracle) developed their database (Oracle Database) that the language took off! [4]

APIs with OracleDB

OracleDB puts no limitations on the type of API you can put on top of it! Actually, Oracle is one step ahead of you and built Oracle REST Data Services (ORDS) to generate modern APIs for your database.

Generating your API with SQL Developer

SQL Developer allows you to generate a table API with stored procedures!

^ SQL Developer a free IDE and DBA console for Oracle Database 
/* Stored Procedures for CRUD API */
/* Generated by SQL Developer for our Accounts table */

CREATE OR REPLACE PACKAGE accounts_tapi IS
    TYPE accounts_tapi_rec IS RECORD ( email_address accounts.email_address%TYPE,
    created_at accounts.created_at%TYPE,
    id accounts.id%TYPE,
    last_name accounts.last_name%TYPE,
    first_name accounts.first_name%TYPE,
    num_legs accounts.num_legs%TYPE );
    TYPE accounts_tapi_tab IS
        TABLE OF accounts_tapi_rec;

-- insert
    PROCEDURE ins (
        p_email_address   IN accounts.email_address%TYPE DEFAULT NULL,
        p_created_at      IN accounts.created_at%TYPE DEFAULT NULL,
        p_id              IN accounts.id%TYPE,
        p_last_name       IN accounts.last_name%TYPE DEFAULT NULL,
        p_first_name      IN accounts.first_name%TYPE DEFAULT NULL,
        p_num_legs        IN accounts.num_legs%TYPE DEFAULT NULL
    );
-- update

    PROCEDURE upd (
        p_email_address   IN accounts.email_address%TYPE DEFAULT NULL,
        p_created_at      IN accounts.created_at%TYPE DEFAULT NULL,
        p_id              IN accounts.id%TYPE,
        p_last_name       IN accounts.last_name%TYPE DEFAULT NULL,
        p_first_name      IN accounts.first_name%TYPE DEFAULT NULL,
        p_num_legs        IN accounts.num_legs%TYPE DEFAULT NULL
    );
-- delete

    PROCEDURE del (
        p_id   IN accounts.id%TYPE
    );

END accounts_tapi;
/

CREATE OR REPLACE PACKAGE BODY accounts_tapi IS
-- insert

    PROCEDURE ins (
        p_email_address   IN accounts.email_address%TYPE DEFAULT NULL,
        p_created_at      IN accounts.created_at%TYPE DEFAULT NULL,
        p_id              IN accounts.id%TYPE,
        p_last_name       IN accounts.last_name%TYPE DEFAULT NULL,
        p_first_name      IN accounts.first_name%TYPE DEFAULT NULL,
        p_num_legs        IN accounts.num_legs%TYPE DEFAULT NULL
    )
        IS
    BEGIN
        INSERT INTO accounts (
            email_address,
            created_at,
            id,
            last_name,
            first_name,
            num_legs
        ) VALUES (
            p_email_address,
            p_created_at,
            p_id,
            p_last_name,
            p_first_name,
            p_num_legs
        );

    END;
-- update

    PROCEDURE upd (
        p_email_address   IN accounts.email_address%TYPE DEFAULT NULL,
        p_created_at      IN accounts.created_at%TYPE DEFAULT NULL,
        p_id              IN accounts.id%TYPE,
        p_last_name       IN accounts.last_name%TYPE DEFAULT NULL,
        p_first_name      IN accounts.first_name%TYPE DEFAULT NULL,
        p_num_legs        IN accounts.num_legs%TYPE DEFAULT NULL
    )
        IS
    BEGIN
        UPDATE accounts
        SET
            email_address = p_email_address,
            created_at = p_created_at,
            last_name = p_last_name,
            first_name = p_first_name,
            num_legs = p_num_legs
        WHERE
            id = p_id;

    END;
-- del

    PROCEDURE del (
        p_id   IN accounts.id%TYPE
    )
        IS
    BEGIN
        DELETE FROM accounts
        WHERE
            id = p_id;

    END;

END accounts_tapi;
